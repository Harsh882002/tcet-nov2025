Clear project briefs for students — each card shows outcomes, required features, one helpful hint, and submission details.

1
Digital Calculator
Buttons for +, −, ×, ÷ with a clean UI and input area.
Learning outcomes
  • Handle click events & inputs
  • Parse expressions or tokens
  • Update DOM with results
Required features
  1. Number & operator buttons
  2. Display area with expression & result
  3. Clear & backspace
Hint: Maintain the current expression as a string and tokenise/evaluate it safely.
Submission: index.html, style.css, script.js + screenshot + short note.

Required features
  1. Show test paragraph
  2. Start/stop timer & calculate WPM
